Online Transcript Management System (OTMS)

Project Overview

The Online Transcript Management System (OTMS) is a secure web-based system developed to manage transcript requests digitally. The system allows students to submit transcript requests remotely while administrators review, approve, and upload authenticated transcripts for secure download.

Problem the System Solves

Africa Zeal University host students from different African countries. After graduation, students may not be able to return to campus due to financial constraints, geographical distance, or personal circumstances. In some cases, transcripts are delayed or not issued during graduation. This creates difficulties for employment, scholarships, and postgraduate applications.

System Objectives

  • Enable remote transcript request submission
  • Provide admin-controlled approval workflow
  • Allow secure upload of authenticated transcripts
  • Enable students to download approved transcripts
  • Provide notification system for transparency

User Roles & Functionalities

Students:

  • Create account (requires admin approval)
  • Submit transcript requests
  • Track request status
  • Download approved transcripts
  • Manage profile & receive notifications

Admin / Registrar:

  • Approve student accounts
  • Approve or reject transcript requests
  • Upload authenticated transcripts
  • Manage records and send notifications

Technologies Used

HTML, CSS, JavaScript, PHP, MySQL (Developed using the Waterfall Model)

Impact

OTMS reduces travel costs, improves transparency, supports e-learning students, and enhances institutional efficiency through structured digital transcript management.
